# Additional setup instructions
Change the .env file to include your MySQL database password:
DB_PASSWORD=your_mysql_password:
and set the database name:
DB_DATABASE=gota_video_db

Check if db/schema.sql exists 
Then login to MySQL:
mysql -u root -p
Then run the following command:
source db/schema.sql;
The above creates an empty database. If you want to add some test data into it, run this: 
source db/seed_data.sql;
